19- Decision coverage: IF ELSE ile ilgili siklar vardi. Bu soru icin asagidaki kisim bilinse yeterli sanirim. 


4.3.3 When 100% decision coverage is achieved, it executes all decision outcomes, which 
includes testing the true outcome and also the false outcome, even when there is no explicit 
false statement (e.g., in the case of an IF statement without an else in the code). Statement 
coverage helps to find defects in code that was not exercised by other tests. Decision 
coverage helps to find defects in code where other tests have not taken both true and false 
outcomes.
